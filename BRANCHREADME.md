This is the main development branch for the Git Graph Visualizer (GGV) project, featuring enhanced visual styling and condensed graph layouts.

## Key Features

- **Condensed Graph Visualization**: Shows only essential commits (referenced commits + junction points)
- **Color-Coded References**: Different colors for local branches, remotes, tags, and special refs
- **Enhanced Typography**: Modern Arial fonts with professional styling
- **Compact Layout**: Optimized spacing for clean, readable output
- **Root Commit Integration**: Automatically includes repository root commits
- **SVG Export**: High-quality scalable vector graphics output
- **Branch README Support**: Displays first line of BRANCHREADME.md files in branch nodes

## Branch Purpose

This master branch contains the stable, production-ready version of GGV with all the latest visual enhancements and functionality improvements.

## Development Workflow

1. Create feature branches for new development
2. Test thoroughly with various repository structures  
3. Merge back to master when stable
4. Tag releases for version management

The tool generates beautiful, compact git graph visualizations that help developers understand repository structure at a glance.
